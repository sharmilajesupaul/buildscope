# Security Policy

## Reporting a Vulnerability

Please do not open a public issue for a security vulnerability.

Instead, contact the repository owner through GitHub:

- https://github.com/sharmilajesupaul

If GitHub Security Advisories are enabled for this repository, prefer a private report there.

## What to Include

- affected version or commit
- steps to reproduce
- impact
- any suggested mitigation

I will review reports as quickly as possible and coordinate a fix before public disclosure when needed.
